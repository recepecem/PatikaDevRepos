## Usage

Projeyi cloneladıktan sonra Visual Studio Code programında açınız.

Linux için:
```linux
cd kodluyoruzilkrepo
code .
```
eğer snap kullanılarak indirildiyse:
```linux
cd kodluyoruzilkrepo
service snapd.apparmor start
/snap/bin/code
```
